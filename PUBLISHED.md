## Run Locally

```bash
./run.sh
```

This installs dependencies (Python venv + npm), kills any existing processes on ports 8000 and 5173, and starts both servers. Backend at `http://localhost:8000`, frontend at `http://localhost:5173`.

**Prerequisites:** Python 3, Node.js 20+, Ollama running `gemma2:latest` on `http://localhost:11434`.

For detailed setup, see [DEPLOYMENT.md](./DEPLOYMENT.md).
